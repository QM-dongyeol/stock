# 주식관리 웹사이트 명세서

## 1. 프로젝트 개요
- **프로젝트명**: Dividend Tracker
- **유형**: 단일 페이지 웹 애플리케이션
- **핵심기능**: 배당금 추적, 투자금 대비 배당률 계산, 추가 매수 시 잔액 관리
- **사용자**: 개인 투자자

## 2. UI/UX 명세

### 레이아웃 구조
- **헤더**: 로고 + 타이틀 (고정)
- **메인**: 카드 기반 2단 그리드
  - 좌측: 주식 입력 폼
  - 우측: 보유 주식 목록 + 요약
- **모달**: 추가매수, 배당입력 모달
- **반응형**: 768px 이하에서 1단 구성

### 컴포넌트

#### 입력 폼 카드

- 주식명 (text)
- 매수가 (number)
- 주식수 (number)

#### 요약 바
- 총 투자금

#### 추가매수 모달
- 주식명 (readonly)

- 주식명 (readonly)
- 배당 지급일 (date)

- 월별 배당 일정

## 3. 기능 명세

### 데이터 모델
```javascript
{
  id: string,
  accountName: string,
  stockName: string,
  purchasePrice: number,
  shares: number,
  totalAmount: number,
  dividendCycle: 'monthly' | 'quarterly' | 'semiannual' | 'annual',
  dividends: [
    {
      date: string,
      amount: number
    }
  ]
}
```

### 핵심 기능
1. **주식 등록**: 새 주식 추가
2. **추가매수**: 기존 주식에 매수 추가 (주식수, 총액 합산)
3. **배당입력**: 각 주식별 배당 기록
4. **배당률 계산**: (연간 배당금 / 투자금) * 100
5. **월 배당금 합산**: 모든 주식의 월 예상 배당금
6. **데이터 삭제**: 개별 주식 삭제

### 계산 로직
- 총액 = 매수가 × 주식수
- 주당 배당액 = 총 배당액 / 총 주식수
- 투자금 대비 배당율 = (연간 총 배당금 / 총 투자금) × 100
- 월 예상 배당금 = 월간 배당은 그대로, 분기는 /3, 반기는 /6, 연간은 /12

## 4. Acceptance Criteria

- [ ] 주식 정보를 입력하여 목록에 추가 가능
- [ ] 추가매수 버튼 클릭 시 모달이 열리고 기존 데이터와 합산
- [ ] 배당입력 모달에서 지급일과 배당액 입력 가능
- [ ] 목록에 주당 배당액과 투자금 대비 배당율 표시
- [ ] 요약 바에 총 투자금, 월 예상 배당금, 평균 배당률 표시
- [ ] 삭제 버튼으로 항목 제거 가능
- [ ] 새로고침 시 데이터 유지 (localStorage)
- [ ] 반응형 레이아웃 정상 작동
- [ ] 애니메이션이 부드럽게 동작
